# Electricity_Bill_Management

## Main Functionality of the App:-  
* For controlling authorized users login.  
* Maintaining users record.  
* Paying elctricity bill from anywhere in a graphically manner.   
* User Friendly.   
* This is a Mini-project developed using Java

## Admin Role as Following:-  
* Register new Consumer.
* View all consumer.
* Add new bills.  
* View all pending payments.  
* View all complains.
* Update status of complain.
* Delete consumer.    

## Users Role as Following :-  
* Login.  
* View bill amount .
* Pay bill amount.  
* Raise a complain.
* View all complains.
